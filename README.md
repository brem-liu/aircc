# aircc                                                                                                                               
                  
  The universal remote for coding agents.                                                                                                   
                  
  Remote into Claude Code, Codex, Gemini CLI, or any terminal tool on your laptop or VPS — and control them from your phone. Encrypted P2P.
  No public IP needed.

  **Preparing for open source. Releasing soon.**

  In the meantime, try it at [aircc.app](https://aircc.app).
